Тестовое задание для MEDODS,
Вариант А
Вариант А (Практический) Консольный справочник

Суть
Написать скрипт, который читает список пользователей из CSV и выводит в консоль отфильтрованный результат.

Что должно быть

1. На вход принимает путь к CSV-файлу через аргумент командной строки
2. CSV содержит колонки: name, email, age
3. Выводит в консоль только пользователей старше 18 лет
4. Если файл не найден или битый — вывести понятную ошибку, не падать
5. Вывод в виде таблицы (можно просто через print с форматированием)
